# Project NOAH Roadmap

Project NOAH is a lifelong journey.

This roadmap represents the planned evolution of NOAH from an AI-powered workflow into a trusted AI companion.

---

# Phase 0 — Foundation

## Goal

Establish the philosophy and architecture of Project NOAH.

### Objectives

- Create GitHub repository
- Define Manifesto
- Define Vision
- Define Principles
- Design Architecture
- Write initial documentation

Status: ✅ Completed

---

# Phase 1 — Infrastructure

## Goal

Build the core infrastructure that powers NOAH.

### Objectives

- Synology NAS setup
- Docker / Container Manager
- PostgreSQL
- n8n
- Ollama
- Open WebUI
- Wake-on-LAN
- Remote access (Tailscale)

Status: 🚧 In Progress

---

# Phase 2 — Memory

## Goal

### Objectives

- Context management

Status: ⏳ Planned
