# Project NOAH

> Building my own Jarvis, one step at a time.

Project NOAH is my lifelong project to build my own trusted AI companion.

The goal is not simply to build another AI assistant.

The goal is to create an AI that grows with me, understands me, remembers my knowledge, protects my data, and helps me throughout my life.

Every system, every model, every workflow, and every piece of infrastructure exists to support that vision.
